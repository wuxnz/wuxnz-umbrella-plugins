About Umbrella

Umbrella is an open-source mobile application for streaming media content. It provides a flexible platform for accessing various media sources through plugins.

GitHub: https://github.com/new-umbrella/umbrella

To install a plugin, click the "Install" button below or paste the plugin URL into the Umbrella app's plugin page.
